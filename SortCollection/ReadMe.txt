== Description ==

SortCollection is a dll with different sorting algorithms. Currently supported:  
* Bubblesort
* Selectionsort 
* Insertionsort 
* Heapsort 
* Mergesort
* Countingsort
* Shellsort with different gap sequences

Example:
var heapSortedList = list.SortWithHeapSort();
